# Interaction Guidelines

ClearMatch interactions should feel quick, warm, and intentional.

## Timing

- Button press feedback: 150-220ms.
- Card hover and route entry: 220-300ms.
- Message entry: 220ms.
- Progress bars: 550ms.

## Profile Cards

Profile cards use:

- Immersive photo-first layout.
- Bottom gradient fade.
- Text overlay for name, age, and city.
- Floating information panel.
- Drag tilt with swipe thresholds.
- Elevated circular actions.

## Chat

Chat supports:

- Message bubble entry animation.
- Subtle timestamps.
- Typing dots animation.
- Scroll anchoring when new messages arrive.
- Composer focus glow.

## Feedback

Every important action should produce visible feedback:

- Success toast for sent messages.
- Error text for blocked or rejected actions.
- Loading states for chat and route transitions.
- Empty states with friendly copy and action buttons.

## Motion Boundaries

Animations should never block interaction, obscure content, or feel theatrical. The product should feel alive, not noisy.
